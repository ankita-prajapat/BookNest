#  BookNest

BookNest is a full-stack online bookstore developed using React, Node.js, Express.js, and MongoDB. The application allows users to browse books, explore categories, manage wishlists, add items to the cart, and place orders. An admin dashboard is also included for managing books, users, and orders.

## Features

* User Registration and Login
* JWT Authentication
* Browse Books by Categories
* Search and Filter Books
* Wishlist Management
* Shopping Cart
* Order Management
* Admin Dashboard
* Responsive Design

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JWT (JSON Web Token)
